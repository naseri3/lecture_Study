// VS Code New terminal 세팅
1. git init                 => 프로젝트 올리기
    // Initialized empty Git repository in 경로 가 나오면 성공!!
2. git add .                => 모든 파일 업로드 하겠다
3. git add index.html       => 파일 한개만 업로드 가능
4. git status               => 상태를 확인하는 용도
5. git commit -m "first commit"         => 히스토리 만들기
    // "first commit"은 히스토리의 이름이다.
6. git remote 복사해서 프로젝트 터미널에 붙이기         => git 허브에 있는 링크(연결고리 생성)
7. git remote -VS           => 주소 확인
8. git push -u origin main
9. git add .    
10. git commit -m "second commit" 
11. git push -u origin main


// main 명령어 변경
